# grot
A solution to grot game

This short script is able to:
1) generate grot sample like below depending on the side you want:
```
sample = [
   ['u', 'd', 'u', 'u'], # ↑ ↓ ↑ ↑
   ['u', 'r', 'l', 'l'], # ↑ → ← ←
   ['u', 'u', 'l', 'u'], # ↑ ↑ ← ↑
   ['l', 'd', 'u', 'l'], # ← ↓ ↑ ←
]
```

2) find a position from which you should start and determine a longest chain.
